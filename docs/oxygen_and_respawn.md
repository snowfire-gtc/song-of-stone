# Кислородное голодание и система возрождения

## Реализованные функции

### 1. Кислородное голодание

**Механика:**
- Когда персонаж полностью погружён в воду (голова и ноги в блоках воды), начинается отсчёт кислорода
- Максимальный запас кислорода: `PLAYER_OXYGEN_MAX = 30` единиц
- Каждая секунда под водой уменьшает кислород на 1 единицу
- При достижении 0 кислорода персонаж начинает получать урон (-1 HP каждую секунду)
- Визуальный эффект: частицы удушья (голубые пузырьки) появляются каждые 0.5 секунды

**Файлы:**
- `src/logic.c` - функция `logic_update()` обрабатывает уменьшение кислорода
- `src/particles.c` - функция `particles_spawn_drowning()` создаёт частицы удушья
- `src/common_game.h` - константа `PLAYER_OXYGEN_MAX`

### 2. Смерть персонажа

**Причины смерти:**
- HP ≤ 0 (от удушья, атак, падения)
- Падение ниже нуля (y < 0)
- Урон от падения с большой высоты (реализуется отдельно)

**Эффекты при смерти:**
- Спавн частиц смерти (30 частиц цвета команды)
- Звуковой эффект `SOUND_PLAYER_DEATH`
- Сброс флага, если персонаж держал его

**Файлы:**
- `src/logic.c` - функция `logic_check_character_death()`
- `src/particles.c` - функция `particles_spawn_death()`
- `src/particles.h` - типы частиц `PARTICLE_DEATH`, `PARTICLE_DROWNING`

### 3. Возрождение у трона

**Механика:**
- Персонаж возрождается у трона своей команды:
  - Синяя команда → `throne_blue`
  - Красная команда → `throne_red`
  - Без команды → центр карты
- Время возрождения зависит от расстояния до трона:
  - Формула: `respawn_time = respawn_time_per_block * max_distance`
  - Минимум: 2 секунды
  - Максимум: 120 секунд
- После возрождения персонаж получает неуязвимость на время возрождения

**Сброс состояния при возрождении:**
- Полное восстановление HP (`PLAYER_MAX_HP`)
- Полное восстановление кислорода (`PLAYER_OXYGEN_MAX`)
- Сброс скорости (vx = 0, vy = 0)
- Сброс анимаций и состояний (щит, атака, прицеливание)
- Сброс флага (если держал)
- Позиция: чуть выше трона (y + 32 пикселя)

**Файлы:**
- `src/logic.c` - функция `logic_respawn_character()`
- `src/common_game.h` - поле `respawn_time_per_block` в `WorldParams`
- `src/logic.h` - объявления функций смерти и возрождения

## Технические детали

### Структуры данных

```c
// В Character (common_game.h)
int hp;                    // текущее здоровье
int oxygen;                // текущий кислород (0-30)
bool is_invulnerable;      // флаг неуязвимости
float invuln_timer;        // таймер неуязвимости
```

### Константы

```c
#define PLAYER_MAX_HP 6
#define PLAYER_OXYGEN_MAX 30
#define DAMAGE_FALL_THRESHOLD 10  // блоков
#define DAMAGE_FALL_PER_STEP 4
#define MAX_FALL_DEATH_HEIGHT 34
```

### Частицы

**Частицы смерти:**
- Количество: 30
- Цвет: SKYBLUE (синяя команда) или MAROON (красная)
- Время жизни: 1.0-1.5 секунды
- Разлетаются в случайных направлениях

**Частицы удушья:**
- Количество: 5
- Цвет: голубой полупрозрачный (100, 200, 255, 180)
- Время жизни: 0.6-1.0 секунды
- Движение: вверх (пузырьки)

## Интеграция с другими системами

### Флаг
- При смерти персонаж теряет флаг (`is_holding_flag = false`)
- Флаг остаётся на месте смерти
- Другой игрок может подобрать флаг

### Звуки
- `SOUND_PLAYER_DEATH` - проигрывается при смерти
- Требует реализации в `sound.c`

### Сетевая игра
- Состояние смерти и возрождения синхронизируется через сервер
- Сервер авторитетен в расчёте времени возрождения
- Клиент отображает таймер неуязвимости

## Пример использования

```c
// В основном цикле обновления
logic_update(world, frame_counter);

// Внутри logic_update для каждого персонажа:
// 1. Обновление кислорода в воде
if (в_воде) {
    ch->oxygen--;
    if (ch->oxygen <= 0) {
        ch->hp--;
        particles_spawn_drowning(&g_particles, ch->x, ch->y);
    }
}

// 2. Проверка смерти
logic_check_character_death(world, ch);

// 3. Если умер -> возрождение
if (ch->hp <= 0) {
    particles_spawn_death(&g_particles, ch->x, ch->y, ch->team);
    sound_play(SOUND_PLAYER_DEATH);
    logic_respawn_character(world, ch);
}
```

## Тестирование

### Сценарии для проверки:

1. **Удушье в воде:**
   - Зайти глубоко в воду
   - Дождаться окончания кислорода
   - Проверить получение урона и частицы

2. **Смерть от удушья:**
   - Остаться под водой после окончания кислорода
   - Проверить смерть, частицы и звук
   - Проверить возрождение у трона

3. **Возрождение с разных расстояний:**
   - Умереть рядом с троном (быстрое возрождение)
   - Умереть далеко от трона (долгое возрождение)
   - Проверить таймер неуязвимости

4. **Потеря флага при смерти:**
   - Взять флаг
   - Умереть (например, от удушья)
   - Проверить, что флаг остался на месте смерти
